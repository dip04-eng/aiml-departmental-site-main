# UEM Kolkata — AI Department Website

## Project Info

Official departmental website for the AI department at UEM Kolkata.

## Getting Started

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Editing the Code

**Edit files locally** using your preferred IDE, or directly on GitHub:

- Navigate to the desired file, click the pencil icon, make changes, and commit.

**Use GitHub Codespaces**

- On the repository page click **Code → Codespaces → New codespace**.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.) by running:

```sh
npm run build
```

and uploading the generated `dist/` folder.
